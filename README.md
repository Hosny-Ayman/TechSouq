\# TechSouq API 



An advanced E-commerce backend solution built with \*\*.NET 8 Web API\*\*, demonstrating Clean Architecture principles, Repository Pattern, and robust Data Layer implementation.



\## Overview

TechSouq is designed to handle core e-commerce functionalities including product management, user authentication, and order processing. The project emphasizes separation of concerns by isolating the \*\*Data Access Layer\*\* Â \*\*Business Layer\*\* \*\*API Layer\*\*.



\##  Tech Stack

\- \*\*Framework:\*\* .NET 8 Core Web API

\- \*\*ORM:\*\* Entity Framework Core (Code-First approach)

\- \*\*Database:\*\* SQL Server

\- \*\*Configuration:\*\* Fluent API (Entity Configurations)

\- \*\*Architecture:\*\* N-Tier / Clean Architecture



\##  Project Structure

\- \*\*TechSouq.API:\*\* Contains Controllers, DTOs, and Service injections.

\- \*\*TechSouq.DataLayer:\*\* Handles Database Context (`AppDbContext`), Migrations, Models, and Entity Configurations.
